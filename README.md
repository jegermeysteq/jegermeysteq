# S₀ / Субъектно-ориентированные системы

Этот репозиторий является частью **рамки S₀**.

S₀ — это не продукт, не платформа и не закрытая теория.
Это минимальное основание для мышления о **субъектах, взаимодействиях, дефицитах, ценности и самоорганизующихся системах**.

Назначение этого репозитория — **исследование, реализация и экспериментирование** с идеями, вытекающими из S₀.

---

## Что такое S₀

**S₀ (Subject Zero / Субъект Ноль)** — это минимальная отправная точка:

* существует несколько субъектов,
* субъекты конечны,
* субъекты взаимодействуют,
* взаимодействия оставляют след,
* следы образуют историю,
* из разрывов и незамкнутостей возникают дефициты,
* ценность появляется только при локальном уменьшении дефицитов.

Всё остальное — рынки, иерархии, интеллект, координация, системы ценностей — рассматривается как **эмерджентные явления**, а не как исходные сущности.

Этот репозиторий **не задаёт каноническую реализацию S₀**.
Он является одной из возможных проекций и интерпретаций.
